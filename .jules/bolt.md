## 2025-06-17 - Heavy script lazy loading and string building optimization
**Learning:** Chrome extension popups have a very limited lifecycle. Loading heavy libraries like `html2pdf.bundle.min.js` (~885KB) on every popup open significantly delays the "interactive" state and wastes memory for users who only want Markdown or HTML exports. Also, while `Array.join()` is often preferred for large string construction, V8 optimizations can sometimes make `+=` competitive; however, `.map().join('')` remains cleaner and typically more efficient for structured data like chat messages.
**Action:** Always check for heavy dependencies that are feature-specific and move them to a lazy-loading pattern. Prefer `Array.map().join('')` for mapping data to strings to improve readability and maintain consistent performance.
